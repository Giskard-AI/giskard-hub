2.1.0 (2025-11-24)
==================

We are releasing a new version of the Hub UI that introduces audit logs, a new task management system, enhanced scans, and improved UI and content.
This helps you manage your quality assurance process, track your scans, and improve the collaboration with your team.

**Tasks & Follow-up Workflow**
   Streamline your quality assurance process with a comprehensive task management system:
   
   * **Create Tasks from Failures** - Create tasks directly from failed evaluations or scans
   * **Assign Owners with Notifications** - Assign task owners with email notifications (opt-out available)
   * **Auto-Draft Bad Tests** - Creating a task from a bad test can automatically mark it as Draft
   * **Hide Noisy Results** - Hide noisy or sensitive eval results while still tracking them via tasks

**Draft Conversations & Datasets**
   Better control over your content lifecycle:
   
   * **Clearer Draft/Published Toggle** - More intuitive toggle with helpful explanations
   * **Draft Filter in Tables** - Draft filter added to dataset tables with clearer status labels
   * **Consistent Exclusion** - Draft conversations are consistently excluded from evaluation runs

**Enhanced Scans**
   Improved scanning capabilities and usability:
   
   * **New Built-in Probes** - Two new built-in probes added to the scanning toolkit
   * **JSON Export** - Export scan results in JSON format directly from the UI
   * **UI Improvements** - Various improvements for better readability and stability
